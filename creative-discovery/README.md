# Creative Discovery — Prototype

A bold, cinematic podcast website for **Creative Discovery**, hosted by Joachim Bekkevold — featuring candid conversations about Business, Investing, and Life in Cambodia.

## How to view

Open `index.html` in a browser. No build step or server required.

## Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Hero with one-click play, topic browse, episodes, host, guests, newsletter |
| Episodes | `episodes.html` | Filterable grid (All / Business / Investing / Life in Cambodia) |
| Episode detail | `episode.html` | Player, show notes, transcript, guest box |
| About | `about.html` | Host story, values, CTAs |
| Guests | `guests.html` | Visual guest wall |
| Cambodia | `cambodia.html` | Evergreen business/investing resource hub |
| Be a guest | `be-a-guest.html` | Guest pitch form |
| Sponsor | `sponsor.html` | Sponsorship tiers + contact form |

- HTML5 / CSS3 (custom properties) / Vanilla JS
- GSAP 3.12.5 + ScrollTrigger (animations)
- Lenis 1.1.14 (smooth scroll)
- Splitting.js (hero text reveal)
- Google Fonts: Space Grotesk + Inter
- Tabler Icons (webfont)
